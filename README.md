# ConsoleTest001

My initial attempt at the Boat Movements Question from TestDome. For more information: https://www.testdome.com/questions/c-sharp/boat-movements/134843

There is a 2-D boolean array which serves as the game 'board' with the valid water positions being true and invalid land positions being false. The boat moves in a set pattern and cannot move accross land to get to a valid water positio. The program is supposed to analyse inputed from and to positions or cooridinates and determine whether the boat is able to move to that psotion or not.
